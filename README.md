# Bot Plugin for Minecraft Paper 1.12.2

## Описание
Плагин создает двух NPC ботов, которые выполняют интерактивную сценку:
- Первый бот смотрит вперед
- Второй бот проходит 1 блок, смотрит вниз
- Первый бот поворачивается на звук
- Второй бот проходит еще 1 блок
- Боты смотрят друг на друга и спауним сердечки

## Команды
- `/spawnbots` - Создать ботов
- `/removebots` - Удалить ботов

## Координаты
- Первый бот: X:310 Y:64 Z:-9
- Второй бот: X:301 Y:64 Z:-9

## Требования
- Paper 1.12.2
- Java 8+

## Установка
1. Скачай готовый `.jar` файл
2. Положи его в папку `plugins`
3. Перезапусти сервер
4. Напиши `/spawnbots`
